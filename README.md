# Federgraph FC91

The Federgraph app is a cross-platform Delphi App, using FMX. Versions of the app are available from the Store.

In this repository I will build-up the Federgraph app from scratch. Each commit will be a step. The steps will be described later. The series of descriptions may become the chapters in a book.

I will start by uploading the first batch of steps, so that the head of repository will contain the demo app, kind of a minimal app. Then I will branch out and build the full app in a branch.
